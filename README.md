fs storage wrapper for cnpm

## Installation

```bash
$ npm install fs-cnpm
```

## API

All the APIs are following [cnpm nfs guide](https://github.com/cnpm/cnpmjs.org/wiki/NFS-Guide).

- `uploadBuffer`: upload a file

- `download`: download file by key

- `remove`: remove file by key
